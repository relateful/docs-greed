The dashboard provides an intuitive way to manage all aspects of your Discord servers, from moderation to automation.

<Info>
Dashboard documentation will not be complete till official release. We do not have a date for this yet.
</Info>

## Key Features

### Server Management
- View and manage all your servers in one place
- Configure server settings and permissions
- Manage roles and channels
- View server statistics and analytics
- Customization of Greed

### Moderation Tools
- Advanced moderation dashboard
- User management and history
- Automated moderation settings
- Log viewer and audit trail

### Automation
- Create and manage custom commands
- Set up automated responses
- Configure welcome messages
- Manage reaction roles

### Analytics
- Server growth statistics
- User activity tracking
- Command usage analytics
- Custom report generation

## Getting Started

1. **Access the Dashboard**
   - Visit [dashboard.greed.best](https://dashboard.greed.best)
   - Log in with your Discord account
   - Authorize Greed to access your servers

2. **Select a Server**
   - Choose the server you want to manage
   - Verify you have the necessary permissions
   - Start configuring your server

3. **Configure Settings**
   - Set up basic server settings
   - Configure moderation tools
   - Set up automation features
   - Customize your dashboard

<Warning>
The dashboard requires the following permissions to function properly:
- Manage Server
- Manage Roles
- Manage Channels
- View Audit Log
- Manage Messages
</Warning>
