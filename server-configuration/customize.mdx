Edit how Greed appears in your server

### Requirements

- Bot must have permissions:
  - Administrator



### Commands

### Name

<CodeGroup>

```bash Syntax
,customize name [name]
```

```bash Example
,customize name lucky star
```

</CodeGroup>

### Avatar


<CodeGroup>

```bash Syntax
,customize avatar [link]
```

```bash Example
,customize avatar https://i.pinimg.com/736x/bb/ba/73/bbba73113d69ba7e62d966e677dd5565.jpg
```

</CodeGroup> 

### Banner


<CodeGroup>

```bash Syntax
,customize banner [link]
```

```bash Example
,customize banner https://i.pinimg.com/1200x/cb/73/44/cb7344a57ed4c85b1e509986cfdc5511.jpg
```

</CodeGroup>

### Reset 

<CodeGroup>

```bash Syntax
,customize reset 
```

```bash Example
,customize reset
``` 

</CodeGroup>
